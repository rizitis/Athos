Thank you for installing digiKam!


We would like to inform you that the Windows version of digiKam works quite well even though some features are still missing.

This installer includes the stand-alone version of digiKam Image Editor named Showfoto.

This installer has been compiled under Windows using VCPKG project.


License


digiKam is an open-source project published under the General Public Licence version 2 or later.


Internationalization


This installer includes lots of application translations. To change the current language, go to the Help menu and use the Switch Application Language option. digiKam needs to be restarted for the change to take effect.


Documentation


To limit size of this bundle, the detailed digiKam documentation is not included in the program. The complete handbook is available online
as HTML or
as EPUB.


Requirements and Limitations


The internal database server requires installation of a local database backend for Windows, Mysql or MariaDB.

Camera auto-detection does not work yet under Windows due to limitations in Solid hardware interface used by digiKam.

Panorama and ExpoBlending tools require installing Hugin runtime components to process files.


Contribute and Help


To report a bug or a wishlist item, please follow the instructions at digiKam project web site.

For all current open bug reports for the Windows release, please check the relevant Bugzilla entries.

NOTE: When you discover an issue or bug, please get a backtrace of the debug statements generated by digiKam while running. Unlike under Linux, this information is printed on a system console only visible with a dedicated application named DebugView.
You can download this tool at this url.
